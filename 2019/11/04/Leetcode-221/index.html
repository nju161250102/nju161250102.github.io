221. Maximal Square

发表于 2019-11-04 | 分类于 leetcode

题目描述

Given a 2D binary matrix filled with 0’s and 1’s, find the largest square containing only 1’s and return its area.

在一个由字符’0’和’1’的矩阵内寻找由1组成的正方形的最大值.

解答

使用动态规划的思想, 对应的 dp 矩阵初始化值均为0. dp 矩阵中值的含义为以该位置为右下角的正方形的边长. 对于原始矩阵中每个为 1 的位置, 递推关系式为 dp(i, j)=min(dp(i−1, j),dp(i−1, j−1),dp(i, j−1)) + 1. 递推结束后, dp矩阵中的最大值即为所求正方形边长的最大值.

由于每次计算仅用到相邻的两行, 所以可以简化为一个一维数组.
