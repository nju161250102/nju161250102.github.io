164. Maximum Gap

发表于 2019-10-28 | 分类于 leetcode

题目描述

Given an unsorted array, find the maximum difference between the successive elements in its sorted form.

Return 0 if the array contains less than 2 elements.

给出一个未排序数组，求排序后相邻元素的最大差值

解答

假设数组中有N个元素，最小值是min，最大值是max，排序后相邻元素的最大差值不会小于[(max - min ) / (N - 1)]（可以使用抽屉原理证明）

使用这个差值构造桶，分类的同时记录每个桶中的最大值和最小值，最大差值出现在有元素的两个桶的前一个最小值与后一个最大值之间

补充：桶排序

将待排序数组中的元素分到有限数量的桶子里。每个桶子再分别排序（有可能再使用别的排序算法或是以递归方式继续使用桶排序进行排序）。
