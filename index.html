Leetcode-208

发表于 2019-10-24 | 更新于 2019-10-25 | 分类于 leetcode

题目描述

Implement a trie with insert, search, and startsWith methods.

实现前缀树这一数据结构。

解答

补充：前缀树

介绍

前缀树是N叉树的一种特殊形式。一个节点的所有子孙都有相同的前缀，也就是这个节点对应的字符串，而根节点对应空字符串。

表示方法

  1. 数组存储子节点。

例如，如果我们只存储含有字母 a 到 z 的字符串，我们可以在每个节点中声明一个大小为26的数组来存储其子节点。对于特定字符 c，我们可以使用 c - 'a' 作为索引来查找数组中相应的子节点。

访问十分快捷方便，但是会造成空间的浪费。

  1. HashMap存储子节点

存储字符到下一个结点的映射。比数组稍慢一点，但是节省了空间。
